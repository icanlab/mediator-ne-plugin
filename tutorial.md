# Developer Tutorial

## Prerequisites

- Python >= 3.6
- Ansible == 2.9

## Install mediator-ne-plugin

Download code:

```
git clone https://github.com/icanlab/mediator-ne-plugin.git
```

Set shell variable `ansible_path` and `ncclient_path`, and run script:

```
cd mediator-ne-plugin
./install.sh
```
